#The difference between Makefile and run_all.sh is that Makefile only runs files that are connected to another file's changes.
#If a file independent of another file, it will not run. Makefile gives a pathway to which files should be altered, and is time efficient in this aspect.
#Using run_all.sh is overkill, and Makefile allows a new collaborator to see how files interact with eacother.
